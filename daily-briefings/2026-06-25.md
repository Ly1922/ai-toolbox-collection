## 🎯 实用工具推荐

### 1. tupper — 本地运行 AI 代码的安全沙箱
> **用途**: 在本地安全沙箱中运行 AI Agent 生成的代码，防止恶意/错误代码影响宿主机
> **亮点**: 开源、本地运行、无需云服务

### 2. opentag — 团队协作 AI Agent
> **用途**: 在 Slack/GitHub 中 @mention Codex/Claude Code，自动处理请求并回复到线程
> **亮点**: 开源、自托管、适合团队引入 AI Agent 低门槛

### 3. codex-adversary — Claude Code 的思维多样性
> **用途**: 在 Claude Code 中引入 Codex 作为自动对抗审查者，互补两种模型思维
> **亮点**: 不依赖外部服务，本地运行，简单脚本即可使用

### 4. browser-search — 自托管 AI Agent 网页搜索
> **用途**: 为 AI Agent 提供去幻觉的网页搜索能力，SearXNG 自托管
> **亮点**: 完全免费、无限量、防幻觉设计
