## 一、今日最值得看

**1. opencode-see-image｜给 OpenCode 非视觉模型"开天眼"**
OpenCode 插件，当用户传截图给纯文本模型时，自动路由到视觉模型生成文字描述后返回。一行安装：`opencode plugin opencode-see-image --global`。适合：用 OpenCode 但模型不支持看图的人。新项目 23⭐，昨天刚发布。
🔗 [GitHub - alfaoz/opencode-see-image](https://github.com/alfaoz/opencode-see-image)

**2. buy-vs-build｜阻止 AI Agent 重复造轮子**
一个跨 Agent 规则集（Codex / Claude Code / Cursor / Gemini 通用），让 Agent 在下笔写代码前先问自己：这个问题是不是能用内置功能、已安装依赖、开源库或商业方案解决？附带 ADR 记录、依赖安全审计、PR 检查。理念很对，实用度高。
🔗 [GitHub - stdin/buy-vs-build](https://github.com/stdin/buy-vs-build)

**3. qiaomu-llm-mcp｜多模型 MCP 网关**
把 DeepSeek、Z.ai、OpenAI、Anthropic 等 Provider 统一成一个 MCP Server。支持单模型调用、多模型对比、流水线、HeavySkill 式多轮讨论。密钥放 macOS Keychain，MCP Server 只暴露"密钥是否存在"不暴露明文。36⭐，两天内新项目。
🔗 [GitHub - joeseesun/qiaomu-llm-mcp](https://github.com/joeseesun/qiaomu-llm-mcp)

**4. GoalPro｜意图放大 × Goal Contract 协议**
面向 Codex / Claude Code 的 Skill。模糊任务→先写 Goal Contract：真实意图、完成标准、证伪条件、暂停条件、非目标。杜绝 Agent"猜你意思、跑通就算赢"。中文文档，可直接装。
🔗 [GitHub - KimYx0207/GoalPro](https://github.com/KimYx0207/GoalPro)

## 四、对亚马逊美工有用

1. **Wan 2.7 Prompt Generator** — 通义万相 2.7 系列的提示词模板/路由/校验工具。支持文生图、图生图、图生视频。如果你在设计工作中用 Wan 系列的模型，这个 Skill 能帮你规范 Prompt 结构。
🔗 [GitHub - Marcolexxx/wan2.7-prompt-generator](https://github.com/Marcolexxx/wan2.7-prompt-generator)

2. **codex2ppt**（刚发布，1⭐）— 基于 AI 生成全图幻灯片来做 PPT。如果你的设计交付需要出"商品场景图+排版"组合效果展示，可以关注。
🔗 [GitHub - Azhi-ss/codex2ppt](https://github.com/Azhi-ss/codex2ppt)

3. **ImageGenBySeed**（火山引擎免费额度）— 一键装到 Codex/Claude Code/Cursor 的图片生成 Skill。火山引擎有免费额度，适合临时需要快速出图试效果。
🔗 [GitHub - ZhangYang2297/ImageGenBySeed](https://github.com/ZhangYang2297/ImageGenBySeed)
