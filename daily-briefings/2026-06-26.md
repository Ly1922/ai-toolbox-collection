## 🚀 本周最值得关注

### 1. OpenKnowledge — AI 原生的 Obsidian/Notion 替代品
- **GitHub:** [inkeep/open-knowledge](https://github.com/inkeep/open-knowledge) ⭐346
- **语言:** TypeScript | **许可:** GPL-3.0
- 一个漂亮的 AI 原生 Markdown 编辑器和 LLM Wiki。登上 HN Show 双榜（199 pts + 另一次展示），是本周最热门的 AI 知识管理工具。支持 2nd-brain、agent skills、知识库构建。
- **推荐理由:** 如果你在用 Obsidian + AI 写笔记，这个项目完全是 AI-native 设计，值得替换尝试。

### 2. motion-skills — 50 个开源 AI Agent 动效技能
- **GitHub:** [iart-ai/motion-skills](https://github.com/iart-ai/motion-skills) ⭐209
- **语言:** HTML | **许可:** MIT
- 50 个开源技能包，教你的 AI 编码助手制作动效/动画/视频：动态排版、数据可视化、解说视频、短视频/Reels、WebGL、Manim。14 个可安装包。
- **推荐理由:** 如果你的 Cursor/Claude Code/Codex 需要生成动效内容，直接装上就用，无需从头写 prompt。

### 3. peerD — 浏览器原生的 AI Agent 运行时
- **GitHub:** [NotASithLord/peerd](https://github.com/NotASithLord/peerd) ⭐111
- **语言:** JavaScript | **许可:** Apache-2.0
- 首个浏览器原生的 AI Agent 运行时 — Chrome/Firefox 扩展，在浏览器内运行 agent 循环：操控标签页、启动沙箱计算（JS Notebook、WASM Linux VM、客户端应用），P2P 分享构建成果。BYOK、无后端、无遥测。
- **推荐理由:** 把 agent 直接跑在浏览器里，不需要服务器部署，理念新颖。
